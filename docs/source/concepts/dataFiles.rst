Data File Types
===============

There are a few ways to store data on the disk.

ASCI Files
----------

This is the most common and intuitive. It uses ASCI characters to store data

delimiters
^^^^^^^^^^

This is the term for the separators. CSV is short for common separated values, the delimiters are commas in this case.


Binary Files
------------

This is less common, but the more efficient way to store data. Instead of storing floating point numbers with a string of ~characters (10 bytes), it stores it as floating point values (4 bytes). This is great but it requires a secondary software to view the data. On the other hand, ASCI files can be opened in any text editor.
