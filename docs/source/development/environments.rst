Enviroments (IDEs)
==================

There are two interactive development environments (IDEs) suggested here.

Spyder
------

To install Spyder see `Spyder's installation page`_.

.. _Spyder's installation page: https://docs.spyder-ide.org/current/installation.html

This IDE has many helpful features and an excellent debugger. It lacks refactoring, however. 


PyCharm
-------

to install PyCharm, see `PyCharm's installation page`_.

.. _PyCharm's installation page: https://www.jetbrains.com/help/pycharm/installation-guide.html#standalone_linux

I haven't used this IDE very much, but it has extensive refactoring capabilities. At first glance, it looks beautiful and it's refactoring is simple. The refactoring alone makes this IDE worth checking out.
