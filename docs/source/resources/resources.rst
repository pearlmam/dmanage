XDSM Diagrams
-------------

XDSM diagrams are used to visualize models, understanding solver loops, and specifying interfaced between models. `OpenMDAO`_ uses it. See `this video`_ for an explanation.

.. _OpenMDAO: https://openmdao.org/newdocs/versions/latest/main.html
.. _this video: https://www.youtube.com/watch?v=yutKRwIL3BA
